# Codex Entry Instructions

This project is an interactive New Concept English 2 courseware system.

When the user says "开始生产 Lesson N" or asks to produce a lesson, first read
`LESSON_PRODUCTION_PLAN.md` in the repository root, then read the matching
per-lesson plan at `lesson_plans/lesson_NNN.md` such as
`lesson_plans/lesson_003.md`.

Do not start from broad batch generation. Lesson production is planned one lesson
at a time, with the course design card as the source of truth.
